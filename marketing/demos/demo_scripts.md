## Demo 1: Email Manager Agent (2 min)

### Script:

**[Screen: Terminal]**

"Let me show you how Cerberus automates email management for businesses."

```bash
python ~/cerberus/demos/demo_email.py
```

**[Show output as it runs]**

"First, the agent categorizes incoming emails automatically - urgent, work, invoices, newsletters. No more manual sorting."

**[Point to categorized emails]**

"It extracts action items from each email. See these tasks? They were pulled automatically from the email content."

**[Point to action items]**

"And it drafts responses for your approval. You review, tweak if needed, and send. What used to take an hour now takes 5 minutes."

**[Show draft response]**

"This agent saves my clients 5-10 hours per week on email alone."

## Recording Tips:

1. **Resolution:** 1920x1080
2. **Audio:** Clear voiceover, no background noise
3. **Pace:** Slow enough to read, fast enough to keep attention
4. **Tool:** QuickTime (Mac) or OBS
5. **Length:** 1.5-2.5 minutes each
6. **Thumbnail:** Terminal with Cerberus logo + agent name
